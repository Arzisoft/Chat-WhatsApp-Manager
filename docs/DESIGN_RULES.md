# Design Rules — ChatChaos WhatsApp Manager

CRITICAL: NO EMOJIS ALLOWED ANYWHERE

This applies to:
- UI design (no emoji icons)
- Documentation
- Code comments
- Error messages
- UI text
- Help text
- Notifications
- Everywhere else

Use text, icons, or symbols instead:
- Instead of: Use:
- [DONE]
- [PENDING]
- [WARNING]
- [ERROR]
- [INFO]
- [SUCCESS]
- [TODO]

---

## Visual Design Standards

Clean, professional, no emoji. All content text-based with proper typography hierarchy.

Use simple icons (SVG/FontAwesome) if needed, not emoji.

---

**All teams:** Follow this rule strictly.
